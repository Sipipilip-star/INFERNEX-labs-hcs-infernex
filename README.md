# INFERNEX-labs-hcs-infernex
# INFERNEX v1.1 – Post Genesis

Toto je vývojová větev systému INFERNEX, obsahující modulární architekturu
pro AIPO inteligenci, LLM gateway, agentní běh, embedder routing a ekonomický
systém tokenového účtování.

## Spuštění API
uvicorn apps.api.main:app --reload
## Struktura
- infernex-core/ – hlavní inteligence
- apps/ – API + dashboard
- docs/ – dokumentace
- tests/ – testy modulů
